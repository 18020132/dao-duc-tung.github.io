# dao-duc-tung.github.io

Personal blog

## Install

### VS Code Extensions

- `npm` by `egamma`
- `prettier` by `Prettier`
- `Run on Save` by `emeraldwalk`

### VS Code Settings

```json
{
  "search.exclude": {
    "**/node_modules": true,
    "**/bower_components": true,
    "**/*.code-search": true,
    "**/docs": true
  },
  "emeraldwalk.runonsave": {
    "commands": [
      {
        "cmd": "npm run build"
      }
    ]
  },
  "editor.formatOnSave": true
}
```
